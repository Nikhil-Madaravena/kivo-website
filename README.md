# Kivo Website

This is the source code for the official Kivo In-Memory Database showcase website and documentation portal.

## Tech Stack
- React
- Vite
- Tailwind CSS v3
- React Router DOM
- Lucide React

## Local Development

To run the website locally:

```bash
cd website
npm install
npm run dev
```

## Deployment
The production build is configured to output to the `docs/` folder, which is structured for GitHub Pages deployment.

To build:
```bash
cd website
npm run build
```
